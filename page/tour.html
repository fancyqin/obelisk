API文档

configs:

steps: [Obj,Obj1,...]

Obj.elem : < string > 该步骤中高亮元素
Obj.html : < string > 该步骤中说明文字
Obj.dir : < string > 说明文字显示位置：共八个方向按地图东南西北缩写表示："N","S","W","E","NW","NE","WS","ES" ，默认"S",即下方。
Obj.css : < object > 附加样式
Obj.offset : < string > 位置调整
Obj.before : < function(resolve){} > Promise
Obj.clear : < function > 在结束或者下一步时调用，主要用于清除 before 和 callback操作产生的影响
Obj.callback : < function > callback

text : {}

按钮文字

zIndex: < Number >

z-index 值，默认10000

auto : < Boolean >

是否自动开始steps
